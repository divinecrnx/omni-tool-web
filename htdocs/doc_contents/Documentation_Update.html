Interface

Description

This option let's you update the youtube-dl dependency.

The current version of youtube-dl you have will be printed out to screen. Then, you have the option of Updating it or returning to the previous menu.

When updating, just take note that it may take a while no matter your internet speed. Once it is done updating, the program will notify you and then you'll be returned to the main menu.
