Interface

Description

Before using the scripts, I advise you to run the setup to make sure all files and folders are present. If a problem exists, a prompt telling you to run Setup.bat will be present in the menu.

When finished, the setup will create a file called Finish.txt to let the setup know that every file and folder is accounted for and the program is ready to use.

Choosing the setup option will do several things:

  1. The setup navigates to the root directory
    • Checks if Finish.txt is already present. If it is, it stops at this check and tells you it has no reason to run
      • If Finish.txt does not exist, then the setup will create the folders 01 Downloads, 02 Gifs, 03 Conversions and 04 Combines
      • If Downloads.txt does not exist, the setup will create it together with Previous Downloads.txt
  2. The setup then navigates to the 00 Dependencies folder
    • If ytdl.exe exists, it renames it to it's original filename youtube-dl.exe
    • It checks if OK.txt is present in the folder. If it doesn't exist, that means one or more dependencies may be missing
      • It then records the missing dependencies and leaves behind txt files that are named after the missing dependencies
      • It then navigates back into the root folder and leaves behind an Issue.txt file that details the issues that it recorded
      • However, if all dependencies are present during this, the setup will then create the OK.txt file and the setup then ends here without a problem
    • If OK.txt is already present in the folder, that means every dependency is accounted for and it will echo an okay check to the screen

Choosing the purge option will do several things:

  1. The setup will notify you of what files and folders it will delete and whether if you're sure you want to proceed
  2. If you proceed, the setup will then delete everything except for:
    • Menu.bat
    • 00 Dependencies
    • 99 Modules

Note: Make sure none of your downloaded files are in any of the deleted folders as they will be permanently lost.
