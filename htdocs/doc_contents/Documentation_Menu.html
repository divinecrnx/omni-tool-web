Interface

Description

For the current version of Omni Tool, the whole program itself is displayed in windows' very own command line using batch files. As such, it may not be the best looking program anyone has ever seen.

Despite it not being the friendliest looking interface, it gets the job done. Which is more than enough for most people looking to just download or convert a few files.

Plus, batch scripts are not as resource demanding, so that is one of the biggest advantages that it has.
