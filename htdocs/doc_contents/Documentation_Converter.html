Description

The converter has a simple purpose; to convert one media filetype into another.

To start, place your file into the 03 Conversions folder. Be sure to take note of your file's name and extension. Then, run the script and input those details plus the filename and extension of your desired output.

The possibilites are:

  • mp4 to mp3
  • mkv to mp4
  • flv to mp4
  • mp3 to wav
  • mp4 to gif
  • ...and more!

To be honest, the only limit is that your file has to be an appropriate media file. You can't convert say, an mp4 file into a docx file. That would be impossible. Though, fun to think about.
