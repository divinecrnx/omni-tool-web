Description

Combine has a pretty niche use. It was mainly used by me to experiment combining an audioless video with an audio file. I decided to include the script for anyone who wishes to modify it to their liking.

To start, place your file into the 04 Combines folder. Be sure to take note of both of your file's name and extension. Then, run the script and input those details plus the filename and extension of your desired output.

Note: This option may get phased out in the near future. It will be included into the modules folder but it won't be an option in the actual program anymore.
